# 配方法：把二次方程变成完全平方

验收题：解方程 `x² + 4x - 5 = 0`

## 第 1 页：课堂演示

教师按以下顺序板书，并在每一步旁解释“为什么”：

1. 移项：`x² + 4x = 5`
2. 两边同加一次项系数一半的平方：`x² + 4x + 4 = 9`
3. 写成完全平方：`(x + 2)² = 9`
4. 开平方：`x + 2 = ±3`
5. 得到答案：`x = 1` 或 `x = -5`

课堂检查：学生应能指出“为什么要加 4”，并知道等式两边必须同时加 4。

## 第 2 页：焦点、总结与练习

教师焦点建议：框选第 2 步 `x² + 4x + 4 = 9`，点击“设为全班焦点”。

学生实时后处理：框选第 2 步，触发“解释当前步骤”。

课后完整总结应覆盖：移项、配方、完全平方、开平方、两个根，以及常见错误“只给左边加 4”。

练习题：

1. `x² + 6x + 5 = 0`
2. `x² - 8x + 7 = 0`
3. 说明配方时为什么要取一次项系数的一半再平方。
