# 11. 最终验收定义

10 月底正式上线前，必须能现场完成：

1. 拿起 AI Pen 在 Capture Surface 上写。
2. 远端 Web Viewer 实时看到笔迹。
3. 结束 session 后回放笔迹过程。
4. 教育场景生成可编辑课后讲义。
5. 商务场景生成待确认行动项 / 决策。
6. 点击 AI 结果可反查原始笔迹区域和时间。
7. Kickstarter 页面清楚说明当前原型、交付计划和风险。
