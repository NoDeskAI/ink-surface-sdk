# 项目规划

## 项目定位

出海墨水屏项目以“AI 标注识别 + 电子纸阅读/手写设备”为产品主线，H2 的规划目标是完成从软件原型、MVP、样机验证到市场推广的完整闭环。项目管理以可量化结果为准，所有阶段都要能回答三个问题：是否有可展示进展、是否产生可验证收益、是否降低下一阶段风险。

## H2 北极星指标

| 指标 | H2 目标 | 追踪方式 |
| --- | --- | --- |
| 可演示闭环 | 9 月底完成最终版样机 + 最终版软件实测 | 演示视频、实测记录、问题关闭率 |
| 市场验证 | 11 月底完成海外推广运营验证 | 落地页访问、线索、访谈、转化意向 |
| 成本边界 | 8 月底形成样机 BOM 和量产成本区间 | 采购记录、供应商报价、替代料方案 |
| 项目收益 | H2 至少形成可复用技术资产、采购资产和运营资产 | 文档库、原型、样机、素材、用户反馈 |

| --- | --- | --- | --- |

## 管理原则

1. 每个月至少产生一个可验证成果，不能只更新文档。
2. 每个阶段必须记录成本、风险、问题和下一步决策。
3. 软件原型、硬件采购、样机验证、市场运营的成果统一回填到项目总看板。
4. 本地 Markdown 为主编辑源，飞书用于协作查看和同步。
