# 项目启动

## 项目背景

出海墨水屏项目面向海外学习、阅读和轻办公用户，核心目标是把电子纸低功耗、长阅读、手写批注的优势，与 AI 标注识别、知识整理、复盘输出结合起来，形成可演示、可验证、可迭代的软硬件一体产品。

当前阶段的重点不是一次性自研完整硬件，而是先用市面低成本墨水屏硬件验证软件闭环，再并行推进硬件选型、采购、样机装配和海外运营准备。

## 启动目标

| 目标 | 第一版正式口径 | 验收方式 |
| --- | --- | --- |
| 方案收敛 | 完成产品、软件、硬件、数据链路和项目管理文档汇总 | 技术方案总览、项目总看板、H2 目标文档可直接用于评审 |
| 软件原型 | 6 月底完成标注、识别、AI 输出、回屏展示的可演示原型 | 本地或 Web 原型可演示一次完整闭环 |
| 采购准备 | 6 月底完成硬件采购字段、视图和首批采购判断口径 | 硬件采购清单可落到飞书多维表格执行 |
| MVP 验证 | 7 月中旬完成市面低成本墨水屏 + 我们软件的第一版 MVP | 能在真实或近真实设备上演示核心流程 |
| 样机验证 | 8 月底完成样机功能验证和装配 | 能完成显示、输入、同步、供电和结构装配验证 |

## 实施策略

1. 软件先行：先把 PDF/文档标注、笔迹识别、OCR、AI 旁注、结果回写链路跑通。
2. 硬件分层：先用市售低成本墨水屏验证，再进入 7 月底硬件选型与结构设计。
3. 数据闭环：所有实验结论进入项目总看板、风险日志、采购清单和阶段复盘。
4. 本地优先：本地 Markdown 是主编辑源，飞书作为协作查看和阶段同步目标。
