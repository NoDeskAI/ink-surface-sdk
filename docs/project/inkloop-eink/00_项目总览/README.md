# 项目总览

## 项目定位

出海墨水屏项目围绕 E-Ink / 电子纸硬件、端侧交互、云端 AI 能力、内容与标注工作流展开。当前文档库用于沉淀前期方案、项目管理资料、会议纪要、里程碑和阶段复盘。

## 当前工作方式

- 本地 Markdown 是主工作区。
- 飞书用于协作、共享、评审和归档。
- 每周会议纪要和阶段复盘要有固定入口，避免信息散在聊天和临时文件中。

## 核心入口

| 文档 | 用途 |
| --- | --- |
| `项目总看板.md` | 每周看项目状态、风险、收益和进展 |
| `2026H2_量化目标与月度追踪.md` | 每月按现金收益、线索、试用、样机、软件进展复盘 |
| `2026H2_目标与里程碑.md` | H2 阶段排期和关键产出 |
| `出海 E-Ink.md` | 飞书根文档导出的当前快照 |

## 下一步维护动作

- 每周更新 `03_会议纪要`。
- 每个阶段结束更新 `04_阶段复盘`。
- 目标或里程碑变化时同步更新 `2026H2_目标与里程碑.md` 和 `2026H2_量化目标与月度追踪.md`。
- 每周更新 `项目总看板.md`。
- 技术方案发生重大变化时追加 ADR 到 `02_项目管理/决策记录.md`。
