## 维护规则

- 新技术决策先写进 `00_入口与决策_6月方案/技术方案总览.md` 的“当前技术主线”。
- 影响范围、排期或成本的决策同步到 `../02_项目管理/决策记录.md`。
- 原始长文档作为专题依据保留；方向收敛写入总览和项目总看板。
- 新增技术文档必须进入对应功能域目录，不再平铺到本目录根层级。
